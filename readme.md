# <img src="assets/icon-128x128.png" width="40" height="40" align="center"> SketchPop: Quick App & Promo Builder

[![WordPress Plugin](https://img.shields.io/wordpress/plugin/v/sketchpop-builder.svg?color=blue)](https://wordpress.org/plugins/sketchpop-builder/)
[![PHP Version](https://img.shields.io/badge/php-%3E%3D%208.1-777bb4.svg)](https://php.net)
[![License](https://img.shields.io/badge/license-GPLv2-green.svg)](http://www.gnu.org/licenses/gpl-2.0.html)

**SketchPop** is a professional-grade popup and modal builder designed for developers and site owners who want a clean, "App-style" interface for their promotions. Created by **Sanjit Chaurasiya**, it is optimized for high conversions and modern web standards.

> **Official WordPress Page:** [wordpress.org/plugins/sketchpop-builder](https://wordpress.org/plugins/sketchpop-builder/)

---

## ✨ Features
* **Media Library Integration:** Upload banners directly through the native WordPress uploader.
* **High-Conversion Promo Mode:** Drive traffic to the App Store, Play Store, or custom URLs.
* **Modern Glassmorphism UI:** Features a premium blur backdrop and a spring-bounce entrance animation.
* **White Label Ready:** Option to hide the "By Sketchleader" credit link in settings.
* **Focus Mode:** Ability to hide the close (X) button for critical announcements.
* **PHP 8.1+ Optimized:** Fully secure with modern null-safe fallbacks.
* **Lightweight:** Pure PHP, CSS, and minimal jQuery. No heavy external libraries!

## 🛠️ Installation
1. Download the [Latest Release](https://github.com/sketchleader/sketchpop-builder/releases).
2. Upload the `sketchpop-builder` folder to `/wp-content/plugins/`.
3. Activate via the **Plugins** menu in WordPress.
4. Configure your settings in the **SketchPop** admin sidebar.
