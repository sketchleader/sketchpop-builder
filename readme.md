**SketchPop** is a distinctive, lightweight WordPress popup and modal builder designed for high conversions and modern performance. Created by **Sanjit Chaurasiya**, this tool allows you to promote mobile apps, special offers, or announcements with a premium "SaaS-style" Glassmorphism interface.

---

### 🚀 Key Features
* **Media Library Integration:** Upload your own promotional banners directly through the WordPress native media uploader.
* **App Install Optimized:** Dedicated fields for titles, descriptions, and CTA buttons to drive traffic to the Play Store or App Store.
* **Modern Glassmorphism UI:** Features a premium blur backdrop and professional spring-bounce CSS3 animations.
* **Full Branding Control:** Option to hide the "By Sketchleader" credit link via settings for a white-label feel.
* **Focus Mode:** Option to hide the close (X) button to ensure user engagement with critical offers.
* **PHP 8.1+ Ready:** Fully compatible with modern server environments using secure null-safe fallbacks.

### 🛠️ Installation
1.  **Download:** Get the latest `.zip` from the [Releases](https://github.com/SketchLeader/sketchpop-builder/releases) section.
2.  **Upload:** Go to your WordPress Admin > Plugins > Add New > Upload Plugin.
3.  **Activate:** Click Activate and find the **SketchPop** menu in your sidebar.
4.  **Configure:** Enter your promo details and click **Update SketchPop Settings**.
